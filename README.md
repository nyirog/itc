Interval tree clock
===================

This is a naive implementation of interval tree clock [1] in erlang.
API documentation hosted on hexdocs [2].

Build
-----

    $ rebar3 compile


Test
----

    $ rebar3 eunit
    $ rebar3 dialyzer


 [1]: http://gsd.di.uminho.pt/members/cbm/ps/itc2008.pdf
 [2]: https://hexdocs.pm/itc/
